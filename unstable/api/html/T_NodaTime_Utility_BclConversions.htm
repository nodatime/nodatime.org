BclConversions ClassNoda Time

Conversion methods which don't naturally fit into any other types - for example, for enums which can't specify any other code. In most cases, conversions to and from BCL types are provided within the type itself - such as ToDateTimeUnspecified  and FromDateTime(DateTime).

Remarks

            Many of the methods within this class could be expressed as extension methods - but currently
            Noda Time always builds against .NET 2. In a future version, there may be multiple build targets,
            allowing these to become extension methods for the builds which use .NET 3.5 and higher.
