ExtendedIsoPattern
Returns an invariant local date/time pattern which is ISO-8601 compatible, providing up to 7 decimal places of sub-second accuracy. (These digits are omitted when unnecessary.) This corresponds to the text pattern "yyyy'-'MM'-'dd'T'HH':'mm':'ss;FFFFFFF".

Thread Safety
When used with a read-only CultureInfo, this type is immutable and instances may be shared freely between threads. We recommend only using read-only cultures for patterns, although this is not currently enforced.
