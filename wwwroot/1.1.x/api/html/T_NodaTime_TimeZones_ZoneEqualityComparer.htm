Remarks
The default behaviour of this comparator is to consider two time zones to be equal if they share the same wall offsets at all points within a given time interval, regardless of other aspects of each ZoneInterval within the two time zones. This behaviour can be changed using the WithOptions(ZoneEqualityComparer..::..Options) method.
